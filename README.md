# ⛅ 미세먼지 알림 서비스



### 1. 어플리케이션 소개

원하는 시/도 지역 별 미세먼지 정보를 제공합니다.



### 2. 주요 기능

- 사용자가 설정한 기본 지역의 미세먼지 정보 제공

- 특정 시/도의 모든 지역 미세먼지 정보 제공

- 원하는 지역의 즐겨찾기 등록 및 해제

- 미세먼지 등급 별 UI 변경



### 3. 업데이트 예정 기능

특정 지역의 미세먼지 정보와 메세지를 함께 전달하는 카카오톡 공유 기능



### 4. 기술 스택

- React / TypeScript

- Redux Toolkit

- React Query

- Tailwind
